# User Stories

## Community Member
- US-1: As a member, I can register with email/password so I can use the app.
- US-2: As a member, I can submit a new issue with photo, category, location, description.
- US-3: As a member, I can view my submitted issues and their status.
- US-4: As a member, I receive notifications when my issue status changes.

## Facility Manager
- US-5: As a manager, I can view all reported issues and filter by status.
- US-6: As a manager, I can assign a worker to an issue.
- US-7: As a manager, I can change priority and status, and close issues.

## Worker
- US-8: As a worker, I see only issues assigned to me.
- US-9: As a worker, I can mark issues IN_PROGRESS or RESOLVED.
- US-10: As a worker, I can upload a completion photo and add comments.

## Admin
- US-11: As an admin, I can list all users.
- US-12: As an admin, I can activate/deactivate users.
- US-13: As an admin, I can grant or revoke roles.
